Contenido Multimedia

Dentro de los elementos que podemos utilizar en un sitio web, se encuentran las imagenes, las cuales se incorporan dentro de las paginas gracias a la etiqueta <img>. Dentro de dicha etiqueta podemos colocar los atributos src y alt que indican donde se encuentra el recurso y una descripcion en caso que este no sea accesible.

Formatos soportados

Si bien, la compatibilidad de los navegadores fue aumentando con el paso del tiempo, en lo que respecta a imagenes. Hoy en dia se aconseja utilizar los siguiente formatos de imagen:

  • JPG Joint Photographic's Expert Group es un formato de imagen mapa de bits, que admite compresion y soporta hasta 16.777.216 colores. Es ideal para el uso de fotografias o galerias.
  • PNG Portable Network Graphics es un formato de imagen, pesado debido a que no admite compresion, pero soporta el uso de un canal alpha (transparencia). Es ideal para imagenes que deban interactuar con otros elementos.
  • GIF Graphics Interchange Format es un formato de imagen ligero, que solamente soporta hasta 256 canales de colores y admite interpolacion. Antiguamente se utilizaba mucho para botones e iconos animados.
  • WEBP Web Picture es un formato de imagen mapa de bits, desarrollado por Google con el objetivo de maximizar la compatibilidad con los navegadores y facilitar la comparticion de contenidos. Posee las mismas caracteristicas que los formatos JPG y PNG.
  • SVG Scalable Vector Graphics es un formato de imagen vectorial, es decir, que su contenido se calcula mediante coordenadas que dibujan lineas y curvas. Es ideal para iconos o elementos graficos de requieran ser constantemente reescalados, ya que no se pixelan.
