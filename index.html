FrontEnd

Desarrollo del lado cliente (Browser), es decir, toda caracteristica o funcionalidad que el navegador puede interpretar o ejecutar sin necesidad de la intervencion de un servidor.
Entre los lenguajes que se manejan del lado del FrontEnd se encuentran los siguientes:

  • HTML
  • CSS
  • JS

HTML

Corresponde a un lenguaje de marcado cuya sigla signica HyperText Markup Language, utiliza etiquetas para identificar los diferentes elementos que existen en una página y la funcion que cumple cada uno dentro de la misma. El orden de los factores es sumamente importante ya que altera el resultado de lo que muestra el navegador.

Las etiquetas tambien poseen atributos, a los que les corresponde un valor definido por el desarrollador. Las mismas se clasifican, de la siguiente manera, segun su disposicion dentro de la pagina:

  • De Bloque: Son aquellas que ocupan todo el ancho de su contenedor, generan salto de linea y se les puede dar tamaño. Funcionan principalmente como contenedores.
  • De Linea: Son aquellas que ocupan el ancho de su contenido, comparten espacio con otros elementos de su tipo, No se les puede dar tamaño y funcionan principalmente como contenido.

CSS

Lenguaje de Hojas de Estilos en Cascada, cuyas sigas corresponden a Cascading StyleSheets. El mismo utiliza selectores y reglas para definir la apariencia de los elementos de la pagina web. A diferencia de HTML, el orden en que definamos las reglas no afectara, significativamente, la estructura.

Entre las caracteristicas mas comunes que podemos definir con este lenguaje se encuentran los siguiente atributos, compartidos por la mayoria de las etiquetas:

  • margin distancia externa que existe entre las etiquetas, la misma no suma tamaño.
  • border linea que rodea a la etiqueta y cuyo valor, por defecto, es cero.
  • padding relleno que existe entre el borde de una etiqueta y su contenido.
  • content corresponde a los elementos que se encuentran dentro de la etiqueta, podemos modificar el contenido mediante atributos como width y height
