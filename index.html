Lenguajes de Programacion

Es un sistema que posee una sintaxis determina cuya finalidad es el desarrollo de aplicaciones, mediante la indicacion de instrucciones. Los mismos se pueden clasificar segun su implementacion o la manera en la que se crearon programas con estos.

Implementacion

  • Escritorio Corresponden a los programas que se instalan sobre el sistema operativo como Windows con C++
  • Movil Programas que funcionan en los dispositivos moviles que utilizan su propio SO como Android o IOS.
  • Web Son aquellos que se ejecutan desde el navegador, independientemente del sistema operativo.
  • Consola Aplicaciones que funcionan desde la linea de comandos y por lo tanto, no poseen interfaz grafica.

Forma de Creacion

  • Interpretado Lenguaje cuyas instrucciones son ejecutadas directamente por un programa, como por ejemplo el navegador con js.
  • Compilado Aquellos que empaquetan el codigo resultante en un archivo ejecutable como c++, o java
  • Transpilado Creados con algun framework o library que utiliza algun lenguaje que luego es traducido y compilado.
